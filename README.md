# Ejercicio-7-html

Este ejercicio implementa un ciclo de navegación donde las imágenes actúan como hipervínculos.

## Características
* **Rutas de Imágenes:** Se utiliza `../` para acceder a las fotos ubicadas en el directorio padre.
* **Ciclo:** Pagina 1 -> Pagina 2 -> Pagina 3 -> Pagina 1.
* **Interactividad:** La etiqueta `<img>` está anidada dentro de un `<a>`.
